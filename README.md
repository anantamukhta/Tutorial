# MII_Inovation_using_GitHub

# How to init git
- echo "# Tutorial" >> README.md
- git init
- git add .
- git commit -m "first commit"
- git remote add origin https://github.com/anantamukhta/Tutorial.git
- git push -u origin master

# How to update git
- git add .
- git commit -am "update"
- git push -u origin master
